## Study Plan

- 유키 히로시의 [JAVA 언어로 배우는 디자인 패턴 입문](https://github.com/michilu/dpsrc_2009-10-10)을 공부한다.
- 매주 챕터 1개를 진도 나간다.
- 발표자는 책의 챕터 1개를 자유형식으로 발표 및 질문한다.
- 일시: 매주 토요일 11:00 - 13:00

## Command

cd ${directory}

javac -Xlint -d ./ ./*.java
